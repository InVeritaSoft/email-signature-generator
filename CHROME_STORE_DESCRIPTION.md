# Chrome Web Store Description

## Short Description (132 characters max)

Create professional email signatures with customizable templates. Generate HTML signatures for all email clients.

## Full Description (1600 characters max)

**Single Purpose: Generate Professional Email Signatures**

This extension has one focused purpose: to help you create and generate professional email signatures. Simply fill in your information, choose a style, and copy the HTML signature to use in any email client.

**Key Features:**

• 9 Signature Styles - Classic, Dark, Gradient, Vertical, Quadrant, Horizontal Logo, Gradient Blue, and Simple variants
• Fully Customizable - Add name, title, photo, company logo, and contact information
• Social Media Links - Include LinkedIn, Facebook, YouTube, and website
• Real-Time Preview - See your signature before using it
• Email-Ready HTML - Works with Gmail, Outlook, Apple Mail, and all major clients
• One-Click Copy - Copy to clipboard instantly
• Image Upload - Upload photos with base64 encoding for email compatibility
• Auto-Save - Data stored locally, synced across devices
• No Account Required - Works offline, completely private

**How It Works:**

1. Click extension icon
2. Fill in your details
3. Choose signature style
4. Upload photo (optional)
5. Preview and copy HTML
6. Paste into your email client

**Privacy:** All data stored locally. No external servers. Works offline. Your information stays private.

Compatible with Gmail, Outlook, Apple Mail, Yahoo Mail, Thunderbird, and all modern email clients.
